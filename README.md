# NLP Course at CUB 2023

**n-grams**:
In this directory you can find how to use n-grams in python via `markovify` library. 

**Semantic search bot**:
Simple telegram-bot designed to search for similar sentences in telegram chats.

* In `semantic_search_bot/STS` you can find readme on STS Problem and how we solved it
* In `semantic_search_bot` there's telegram bot in `main.py`
*Keep in mind*: you need to add `semantic_search_bot/token.json` with your telegram bot token to run it.
